# orika-mapper
Encapsulation Bean Mapper base on Orika

# Dependencies

    compile 'ma.glasnost.orika:orika-core:1.4.6'

# Usage

Add Spring Scan Packge : com.kuafu.framework.orika

Auto Injection: 

    @Resource
    Mapper mapper;

# Junit Test Case

    com.kuafu.framework.orika.OrikaMapperTest
